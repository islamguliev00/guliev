## Описание

Movie Library — это десктопное приложение на Python с графическим интерфейсом (GUI), созданное с использованием библиотеки `tkinter`. Приложение позволяет:

* добавлять фильмы в библиотеку, указывая название, жанр, год выпуска и рейтинг (от 0 до 10);
* просматривать все добавленные фильмы в виде таблицы;
* фильтровать фильмы по жанру (выпадающий список) и по году выпуска;
* сбрасывать фильтры для отображения полного списка;
* сохранять данные о фильмах в файл `movies.json` между запусками программы;
* загружать сохранённые данные при запуске приложения.

Приложение включает валидацию вводимых данных для обеспечения корректности информации.

## Установка и запуск

### Требования

* Python 3.6 или выше.
* Стандартные библиотеки: `tkinter`, `json`, `os`, `datetime` (устанавливаются вместе с Python).

### Инструкция по запуску

1. Сохраните код приложения в файл с именем `Movie_Library.py`.
2. Убедитесь, что у вас установлен Python.
3. Откройте терминал (командную строку) в папке, где находится файл `Movie_Library.py`.
4. Запустите приложение командой:
   ```bash
   python Movie_Library.py
При первом запуске будет автоматически создан файл movies.json для хранения данных.

Примеры использования
1. Добавление фильма
Заполните поля в интерфейсе:
Название: «Интерстеллар»
Жанр: «Фантастика»
Год выпуска: «2014»
Рейтинг (0–10): «8,6»
Нажмите кнопку «Добавить фильм».
Фильм появится в таблице ниже. Данные также сохранятся в файл movies.json.

2. Фильтрация по жанру
Убедитесь, что в библиотеке есть фильмы разных жанров.
Выберите нужный жанр из выпадающего списка «Фильтр по жанру» (например, «Драма»).
Таблица обновится, отображая только фильмы выбранного жанра.

3. Фильтрация по году
Введите год в поле «Фильтр по году» (например, 2020).
Нажмите кнопку «Фильтровать по году».
В таблице останутся только фильмы, выпущенные в указанном году.

4. Сброс фильтров
Нажмите кнопку «Сбросить фильтры».
Таблица обновится и покажет полный список всех фильмов в библиотеке.
Тестирование
Ниже приведены примеры тестов для проверки корректной работы приложения.

Тест 1: Успешное добавление фильма
Ввод: Все поля заполнены корректными данными (например, название — «Крёстный отец», жанр — «Криминал», год — 1972, рейтинг — 9,2).
Ожидаемый результат: Фильм появляется в таблице. В логах или файле movies.json фиксируется новая запись. Ошибок не возникает.

Тест 2: Проверка валидации — пустое название
Ввод: Поле «Название» оставлено пустым, остальные поля заполнены корректно.
Ожидаемый результат: Появляется окно с ошибкой «Название и жанр обязательны!». Фильм не добавляется в таблицу.

Тест 3: Проверка валидации — некорректный год
Ввод: В поле «Год выпуска» введено значение abc или 3000 (если текущий год — 2024).
Ожидаемый результат: Для abc — ошибка «Год должен быть числом!». Для 3000 — ошибка «Год должен быть от 1888 до [текущий год]». Фильм не добавляется.

Тест 4: Проверка валидации — некорректный рейтинг
Ввод: В поле «Рейтинг» введено значение -5 или 15.
Ожидаемый результат: Ошибка «Рейтинг должен быть от 0 до 10!». Фильм не добавляется.

Тест 5: Работа фильтров
Ввод: Добавить несколько фильмов разных жанров и годов. Применить фильтр по одному из жанров, затем по одному из годов.
Ожидаемый результат: Таблица корректно отображает только те фильмы, которые соответствуют выбранным критериям. После нажатия «Сбросить фильтры» показывается полный список.

Тест 6: Сохранение и загрузка данных
Ввод:
Добавить один или несколько фильмов.
Закрыть приложение.
Запустить приложение снова.
Ожидаемый результат: При запуске приложение загружает данные из movies.json, и все ранее добавленные фильмы отображаются в таблице.

Структура проекта
Movie_Library.py — основной файл с кодом приложения.
movies.json — файл для хранения данных о фильмах (создаётся автоматически).
